Features

Billing, plans, UI themes, and integrations — purpose-built for hosting providers and self-hosters.

Billing & Payments

Collect payments your way.

M12Labs ships with first-class integrations for the payment providers that hosting businesses actually use. Configure gateways globally or per-plan, and track revenue through built-in billing overviews and analytics.

  • ✦
    Multi-provider support

    Stripe, Mollie, and additional gateways — swap or run them side-by-side.

  • ✦
    Checkout wizard

    A rebuilt step-by-step checkout flow guides customers through plan selection, configuration, and payment.

  • ✦
    Billing overviews & analytics

    At-a-glance dashboards surface revenue trends, active subscriptions, and plan performance in one view.

  • ✦
    Per-plan gateway routing

    Route different plan tiers to different gateways for fee optimisation.

Plans & Cycles

Precision pricing for every model.

Whether you bill monthly, quarterly, or by the day, M12Labs handles the math — including node multipliers, promo windows, and free tiers — without requiring any code changes.

  • ✦
    Day-based billing

    Prorate charges precisely for mid-cycle upgrades, downgrades, and cancellations.

  • ✦
    Node-multiplier pricing

    Scale plan costs automatically based on the number of game server nodes provisioned.

  • ✦
    Promo & free tiers

    Create time-limited promotions or permanent free plans — all managed from the UI.

  • ✦
    Plan overrides

    Grant individual users custom rates or cycle durations without affecting the global plan.

  • ✦
    Multi-cycle management

    Run monthly, quarterly, and annual cycles simultaneously on a single panel.

UI & Themes

A panel that looks like yours.

M12Labs features an overhauled interface inspired by jexpanel — polished, fast, and fully themeable so hosting providers can match the panel to their brand without touching code.

  • ✦
    Overhauled UI

    A ground-up redesign inspired by jexpanel delivers a cleaner, faster experience across every screen.

  • ✦
    Live theme editor

    Adjust colours, typography, and layout breakpoints and preview changes immediately.

  • ✦
    Dark-first design system

    High-contrast defaults with consistent visual rhythm across light and dark variants.

  • ✦
    Extensible components

    Drop in custom views and widgets at defined extension points without forking core.

  • ✦
    Responsive layouts

    Mobile-first grid adapts cleanly from phone to widescreen without media-query hacks.

Integrations

Mods and plugins, without the hassle.

M12Labs connects directly to the content platforms your players already use — no manual file transfers, no SSH sessions. Browse, install, and update from inside the panel.

  • ✦
    CurseForge

    Search and install modpacks, mods, and resource packs from CurseForge without leaving the panel.

  • ✦
    Modrinth

    Access the Modrinth mod ecosystem — filter by loader, version, and category in one place.

  • ✦
    SpigotMC

    Locate and install Spigot plugins directly, keeping server software in sync with your chosen builds.
